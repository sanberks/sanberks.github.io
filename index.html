Projects

  • Master Thesis: Beyond AdaGrad: Adaptive Methods for Non-Euclidean Optimization (LIONS, EPFL, 2025)

    Investigating norm constrained linear minimization oracles (LMOs) and developing an optimizer adaptive to the non-euclidean geometry of the loss landscape, with convergence guarantees for nonconvex optimization, specifically targeting deep learning trainings.

  • MSc Semester Project: Exploration in deep imitation learning (LIONS, EPFL, 2024)

    Extending an exploration bonus computation utilized in the linear function approximation to the nonlinear function approximation setting, more specifically neural networks. As the baseline method, a Soft Actor-Critic based imitation learning algorithm is used with the addition of exploration bonus.

  • BSc Senior Project: A Phase Unwrapping Algorithm for Magnetic Resonance Imaging (MRI) Applications (Bogazici University, 2022)

    Design of a segmentation based path-following algorithm for phase unwrapping of MR images as the first step of Quantitative Susceptibility Mapping (QSM), funded by The Scientific and Technological Research Council of Turkey, TUBITAK.

  • Course Project: Dataset Based Imitation Learning (Reinforcement Learning Course, EPFL, 2024)

    Theoretical comparison of three dataset-based imitation learning algorithms: Online Apprenticeship Learning (OAL), Model-based Transition-aware Adversarial Imitation Learning (MB-TAIL), and Imitation Learning via Adversarial Reinforcement Learning (ILARL).
    References: "Lior Shani, Tom Zahavy, and Shie Mannor. Online apprenticeship learning, 2021", "Tian Xu, Ziniu Li, Yang Yu, and Zhi-Quan Luo. Provably efficient adversarial imitation learning with unknown transitions, 2023", "Luca Viano, Stratis Skoulakis, and Volkan Cevher. Better imitation learning in discounted linear MDP, 2024.

  • Course Project: Reproducibility Challenge: Multi-Task Learning using Uncertainty (Fundamentals of Inference and Learning, EPFL, 2024)

    Reproduction and analysis of "Multi-task learning using uncertainty to weigh losses for scene geometry and semantics" (CVPR 2018). Theoretical and implementational aspects examined in light of experimental results.
    Reference: "Kendall, Alex, Yarin Gal, and Roberto Cipolla. "Multi-task learning using uncertainty to weigh losses for scene geometry and semantics." Proceedings of the IEEE conference on computer vision and pattern recognition. 2018."

  • Course Project: Pose Memory Game for Android Devices (Android Development Course, EPFL, 2024)

    Development of an Android game using pose estimation and smartwatch integration. Users mimic yoga poses within a time constraint; scoring is based on pose accuracy and response time. Included real-time database management.

  • Course Project: Object Tracking on RGB-D Video Frames (Machine Vision Course, Bogazici University, 2022)

    Detected horizontally flat surfaces were filtered out, and the remaining objects tracked across frames using PCL and OpenCV in C++. Focused on RGB-D data analysis and 3D object segmentation.
